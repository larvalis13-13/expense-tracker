# 💰 Expense Tracker - Веб-трекер расходов

Простое и удобное веб-приложение для учёта личных финансов. Позволяет отслеживать расходы по категориям, просматривать историю операций и экспортировать данные.

## 🚀 Возможности

- ✅ **Учёт расходов** — добавление трат по категориям
- 📊 **Статистика** — просмотр трат за сегодня, неделю и общую сумму
- 📋 **История операций** — последние 20 транзакций с возможностью удаления
- 📥 **Экспорт в CSV** — выгрузка всех данных для анализа в Excel
- 🎨 **Современный дизайн** — адаптивный интерфейс на Bootstrap 5
- 💾 **База данных SQLite** — простое хранение данных без настройки сервера
